# Neo6502-Kermit

Based on E-Kermit v1.8

## Kermit communication channel

* Neo6502 UEXT UART
* Default speed: 230400bps
* 8-bit, no parity

## Requirement

* [LLVM-MOS SDK](https://github.com/llvm-mos/llvm-mos-sdk/)
